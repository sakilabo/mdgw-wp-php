# MD Gateway for WordPress (mdgw-wp)

A PHP web application for sharing a WordPress site with AI. It is not a plugin: it fetches from the outside via the WP REST API and outputs a table of contents for all posts plus the Markdown of each post. See README.md for an overview.

- Product name (prose / headings): **MD Gateway for WordPress**
- Repository: **`mdgw-wp-php`** / Composer: `sakilabo/mdgw-wp` (no `-php` suffix — Packagist is PHP-only)
- Deploy path (example): **`mdgate`** → `/mdgate/`
